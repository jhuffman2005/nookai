# NookAI

AI-powered interior design visualization. Upload a photo of your space, describe your style, and get a personalized redesign concept with AI-generated imagery.

## Setup

1. Clone the repo
2. `npm install`
3. Add your `REPLICATE_TOKEN` to `.env.local`
4. `npm run dev`

## Deploy to Vercel

1. Push to GitHub
2. Import repo in Vercel
3. Add `REPLICATE_TOKEN` environment variable in Vercel settings
4. Deploy
